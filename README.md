# chat-mockup

Animated mobile chat interaction built with React + Vite + [Motion](https://motion.dev).

A phone mockup that plays a scripted chat conversation, typing indicators and message bubbles animate in with spring physics, then loops.

## Run it

```bash
bun install
bun dev
```

Open the printed URL in your browser.

## Stack

- React 19 + TypeScript
- Vite 8
- Motion 12 (`motion/react`)
